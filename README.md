# Headset-Router-Filter
Utilizing an old Arctic Nova 7 Gen 1 Headset shell, its push button, and its Li-ion battery we replace the PCB with an ESP32 to act as a travel router with a dashboard. It has its own NAT access point, DNS filtering, DoH on cloudflare over TLS on port 443 instead of 53, live web dashboard, captive portal detection, MAC cloning, and more.
